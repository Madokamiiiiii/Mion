Mion - AWC Tools

{% load static %}

Mion is an open-source Django web application that helps manage Anime Watch Club and Manga Reading Club challenges. As of right now, it only is able to handle Monthly and Genre AWC challenges. Mion is able to make challenge forum posts for you, edit them, and delete them. No more having to parse through a challenge code manually to add an anime to a requirement.

AWC Challenge Tool MRC Challenge Tool

Requires an Anilist account. Source code is available on GitHub.
